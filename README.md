## Run Locally

Clone the project

```bash
  git clone https://github.com/basitanwer/blog-v1.git
```

Go to the project directory

```bash
  cd blog-v1
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** React, Next.js, TailwindCSS
